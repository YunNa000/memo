### 프로젝트 생성 
노드js 와 vscode를 설치 후,
작업용 폴더 만들고, 터미널을 열기


```python 
npx create-react-app 프로젝트명
```

npm start 를 통해 터미널에서 미리보기 할 수 있음

라이브러리를 다 모아놓은 플랫폼 : npm </br>
nodejs를 통해 npm 자동 설치 가능

node_modules
public
src 폴더 : 코드 짜는 곳

index.html

app.js  


pachage.json : 프로젝트 정보 (ex. 라이브러리 버전, 자동생성)
